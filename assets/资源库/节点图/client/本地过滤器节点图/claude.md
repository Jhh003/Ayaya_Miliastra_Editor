## 目录用途
- 存放“本地过滤器节点图”代码文件，用于在客户端侧为各类组件（如命中检测、选项卡等）提供布尔或整数形式的判定逻辑。

## 文档约定（重要）
- 这里只描述目录用途与规范，不记录具体资源清单/不解释单个资源是做什么的
- 本 `claude.md` **只描述目录用途与通用编写约束**。
- **不在此列举本目录下具体有哪些节点图文件**，也不解释单个节点图的业务含义。

## 当前状态
稳定可用。该类节点图通常用于 UI 的可见性/可用性判定等“本地过滤”场景（不在本文档维护清单）。

## 注意事项
- 本目录中的节点图均使用 `graph_type: client`，`folder_path` 建议以 `"本地过滤器节点图"` 开头，便于在编辑器中过滤与引用。
- 过滤器图通常以“布尔返回值”为产出，建议使用正向条件 + else 早退的写法保持主流程从 `True` 分支接续，避免在 `if` 条件中直接调用【逻辑非运算】导致校验失败与 UI 阅读负担。
- 若从自定义变量中读取到“字典类数据”（如资源与货币映射），应将承接变量显式注解为 `"泛型字典"`（或更具体类型），以匹配【查询字典长度】【以键查询字典值】等字典节点的端口类型约束。
- 本文件不记录修改历史，只描述本目录的用途、当前状态与使用注意事项。
