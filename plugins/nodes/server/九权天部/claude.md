# 九权天部 Roguelike 核心机制节点目录

## 目录用途
- 九权天部 Roguelike 游戏系统的核心机制节点实现
- 包含三大流派的资源管理与效果触发节点：
  - **钦天监 - 因果命师**：因果链追踪系统（因果点 CP）
  - **鸿胪寺 - 情绪织者**：情绪频谱共振系统（愤怒、恐惧、绝望）
  - **天工部 - 物质匠师**：物质属性编程系统（物质核心 MC）

## 设计原则
1. **简单直观**：初始核心机制易于理解和实现
2. **模块化架构**：支持横向扩展（新能力、效果）和纵向深化（数值成长、机制组合）
3. **低耦合设计**：各系统独立运行，便于单独修改和扩展
4. **可组合性**：机制间可产生有趣的交互，鼓励多样化玩法
5. **平衡基础**：核心机制自带平衡约束，后续扩展有稳定基础

## 当前状态
- 由静态注册器纳入 `plugins.nodes.server`
- 节点函数均使用 `@node_spec` 描述端口，依赖共用 helper 位于 `plugins/nodes/shared`

## 注意事项
- 保持文件名与节点名称一致，便于查找与生成文档
- 不要在节点里导入 UI 层或自动化模块，仅依赖 `engine` 与 `plugins.nodes.shared`
- 输出日志统一通过 `engine.utils.logging.logger`
